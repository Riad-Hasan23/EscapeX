# Sprite Assets — Placeholder Directory

Place your custom PNG/SVG sprite files here. The game currently renders
everything with programmatic shapes and colors, so no images are required
to run. To upgrade a sprite, save the file here and load it in `game.py`.

## Expected Files (optional)

| File              | Size     | Used For                     |
|-------------------|----------|------------------------------|
| `player.png`      | 28×28 px | The X character               |
| `monster_0.png`   | 28×28 px | Rex (Red monster)             |
| `monster_1.png`   | 28×28 px | Blaze (Orange monster)        |
| `monster_2.png`   | 28×28 px | Surge (Cyan-Green monster)    |
| `monster_3.png`   | 28×28 px | Hex (Purple monster)          |
| `cell_mud.png`    | 30×30 px | Mud tile overlay              |
| `cell_slipper.png`| 30×30 px | Slipper tile overlay          |
| `cell_water.png`  | 30×30 px | Water tile overlay            |
| `cell_freeze.png` | 30×30 px | Freeze tile overlay           |
| `wall.png`        | 30×30 px | Wall tile                     |

All images should have transparent backgrounds where applicable.
